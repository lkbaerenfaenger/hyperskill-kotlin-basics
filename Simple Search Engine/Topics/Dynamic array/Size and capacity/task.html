Size and capacity

Assume that for a dynamic array of size \(2\) and capacity \(4\) we perform the following sequence of operations:

add 7
add 1
add 3
insert 4 3
add 6
add 7
add 8

Here "add x" means "add an element x to the end of the array" and "insert x i" means "add an element x at the specified index i".

Once all operations are performed, what will be the difference between the capacity of the new array (new_capacity) and its size (new_size)? Print this number, i.e. new_capacity - new_size in the field below.

Assume that the scaling factor for the array is $2$. Also note that every time you increase the array's capacity, it becomes twice as big as the capacity of the array at the previous step, rather than the initial array's capacity.
