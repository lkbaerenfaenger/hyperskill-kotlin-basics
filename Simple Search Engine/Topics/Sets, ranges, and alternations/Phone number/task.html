Phone number

You are going to check if the regional part of a phone number in some nonexistent country is valid. The requirements are stated as follows:

  1. The regional part consists of 3 symbols.
  2. It should start with a digit, but the digit should not be equal to 7, 8, 9, or 0.
  3. The second symbol should be a digit.
  4. The third symbol should be a part of a hexadecimal notation: that is, it should be either a digit or an uppercase letter in the range from A to F.
